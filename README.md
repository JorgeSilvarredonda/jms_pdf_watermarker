# jms_pdf_watermarker

A simple software to watermark pdfs.

**Dependencies:**
  - kivy
  - PyPDF2

**Instructions:**
  - Just open the program
  - Select the 2 files
  - Indicate where the new .pdf file is going to be located
  - Press "Merge" Thats it!
